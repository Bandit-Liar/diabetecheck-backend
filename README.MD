# DiabeteCheck ‚Äî Backend API v3

Backend RESTful API untuk sistem deteksi dini risiko diabetes berbasis AI.
Dibangun dengan **Node.js + Express.js** sebagai **proxy** ke server FastAPI Garrand
yang menjalankan model TensorFlow/Keras dengan **17 fitur CDC BRFSS 2015**.

---

## Arsitektur Sistem

```
Frontend (port 5173)
    ‚îÇ  POST /api/predict ‚Äî 17 field langsung di root body
    ‚ñº
Express.js Backend (port 5000)   ‚Üê kamu di sini
    ‚îÇ  POST /predict ‚Äî dibungkus: { features: {...}, use_optimal_threshold: true }
    ‚îÇ  Header: ngrok-skip-browser-warning: true
    ‚ñº
FastAPI Garrand (ngrok URL)
    ‚îÇ  Model: diabetecheck_model.keras + scaler_final_17fitur.pkl
    ‚ñº
Express (konversi format response)
    ‚îÇ  { success, data: { prediction, probability, risk_level, ... } }
    ‚ñº
Frontend (tampilkan hasil)
```

## Instalasi

### 1. Install dependencies

```bash
cd backend
npm install
```

Pastikan `axios` sudah terinstall (cek di `node_modules/axios`).
Jika belum, jalankan:

```bash
npm install axios
```

### 2. Buat file `.env`

```bash
copy .env.example .env
```

### 3. Isi `AI_API_URL` di `.env`

Minta URL ngrok terbaru dari **Garrand (AI Engineer)**. URL berbentuk:
```
https://xxxx-xxxx.ngrok-free.app
```

Isi di `.env`:
```env
PORT=5000
NODE_ENV=development
FRONTEND_URL=http://localhost:5173
AI_API_URL=https://xxxx-xxxx.ngrok-free.app
```

> ‚öÝÔ∏è **Penting:** URL ngrok **berubah setiap kali** Google Colab Garrand di-restart.
> Setiap kali Garrand restart Colab, minta URL baru dan update `AI_API_URL` di `.env`.

### 4. Verifikasi FastAPI Garrand aktif

Buka browser dan akses:
```
https://xxxx-xxxx.ngrok-free.app/health
```
Harus muncul response `{ "status": "ok" }`. Jika tidak muncul, berarti server Garrand belum aktif ‚Äî hubungi Garrand.

### 5. Jalankan server

```bash
# Development (auto-restart)
npm run dev

# Production
npm start
```

## API Endpoints

### `GET /api/health`
Cek status server Express.

**Response 200:**
```json
{
  "status": "ok",
  "message": "DiabeteCheck API is running",
  "timestamp": "2026-05-30T10:00:00.000Z"
}
```

---

### `POST /api/predict`
Prediksi risiko diabetes ‚Äî **17 field CDC** langsung di root body.

**Request Body:**
```json
{
  "HighBP": 1,
  "HighChol": 1,
  "CholCheck": 1,
  "BMI": 28.5,
  "Smoker": 0,
  "Stroke": 0,
  "HeartDiseaseorAttack": 0,
  "PhysActivity": 1,
  "Fruits": 1,
  "Veggies": 1,
  "HvyAlcoholConsump": 0,
  "GenHlth": 3,
  "MentHlth": 0,
  "PhysHlth": 0,
  "DiffWalk": 0,
  "Sex": 0,
  "Age": 7
}
```

**Aturan validasi 17 field:**

| Tipe   | Field                                                                                     | Nilai Valid  |
|--------|-------------------------------------------------------------------------------------------|--------------|
| Biner  | HighBP, HighChol, CholCheck, Smoker, Stroke, HeartDiseaseorAttack, PhysActivity, Fruits, Veggies, HvyAlcoholConsump, DiffWalk, Sex | 0 atau 1 |
| Float  | BMI                                                                                       | 10 ‚Äì 100     |
| Skala  | GenHlth                                                                                   | 1 ‚Äì 5        |
| Skala  | MentHlth, PhysHlth                                                                        | 0 ‚Äì 30       |
| Skala  | Age                                                                                       | 1 ‚Äì 13       |

**Response sukses 200:**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "prediction": "Berisiko",
    "probability": 0.72,
    "risk_level": "Risiko Tinggi",
    "input_received": { "...17 field..." },
    "timestamp": "2026-05-30T10:00:00.000Z"
  }
}
```

**Response error validasi 400:**
```json
{
  "success": false,
  "error": "Validation error",
  "details": [
    "Field 'HighBP' harus bernilai 0 atau 1 (diterima: 5)",
    "Field 'BMI' harus berupa angka antara 10 dan 100 (diterima: 999)"
  ]
}
```

**Response error AI service 503:**
```json
{
  "success": false,
  "error": "AI service tidak dapat diakses. Pastikan server model aktif."
}
```

## Troubleshooting

| Error | Penyebab | Solusi |
|-------|----------|--------|
| `AI_API_URL belum diatur` | `.env` belum diisi | Isi `AI_API_URL` di `.env` dengan URL dari Garrand |
| `AI service tidak dapat diakses` | FastAPI Garrand tidak aktif / URL salah | Hubungi Garrand, minta Colab dijalankan ulang dan kirim URL baru |
| `AI service timeout` | Model lambat merespons | Coba lagi, atau hubungi Garrand untuk cek Colab |
| `EADDRINUSE port 5000` | Port dipakai proses lain | Ganti `PORT=5001` di `.env` |
| `Cannot find module 'axios'` | Axios belum terinstall | Jalankan `npm install axios` |

## Catatan Tim

- File `.env` **tidak di-commit** ke GitHub ‚Äî bagikan `AI_API_URL` via chat tim
- **URL ngrok berubah** setiap Colab Garrand restart ‚Äî selalu update `.env`
- History prediksi bersifat **in-memory** ‚Äî hilang saat server Express restart (by design untuk MVP)
- Untuk cek FastAPI langsung: buka `{AI_API_URL}/docs` di browser ‚Üí Swagger UI interaktif
